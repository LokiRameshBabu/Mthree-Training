## ðŸ”„ Loops in Python
Loops allow us to repeat a block of code.

### Types of Loops:
- `for` loop
- `while` loop

## ðŸ“š Dictionaries in Python
A dictionary is a collection of key-value pairs.

### Basic Commands:
```python
my_dict = {"name": "Loki", "age": 22}
print(my_dict["name"])
my_dict["city"] = "Vizag"
del my_dict["age"]
```

## ðŸ“š Sets in Python
A set is a collection of unordered, unique elements.

### Basic Commands:
```python

my_set.add(4)
my_set.remove(2)
```

## âœ… Activity 15: Sum of primes
```python
def is_prime(n):
    if n < 2: return False
    for i in range(2, int(n**0.5)+1):
        if n % i == 0:
            return False
    return True

num = int(input("Enter number: "))
found = False
for i in range(2, num//2+1):
    if is_prime(i) and is_prime(num - i):
        print(f"{num} = {i} + {num - i}")
        found = True
if not found:
    print("Cannot be expressed as sum of two primes.")
```

### âœ… Activity 7: Use `price` and `quantity` dictionaries to calculate total cost

    "milk": 4,
    "butter": 2,
    "crackers": 1.5,
    "rice": 2.25,
    "spaghetti": 1.75,
    "dish soap": 3.25
}

    "milk": 1,
    "butter": 1,
    "crackers": 3,
    "rice": 2,
    "spaghetti": 5,
    "dish soap": 1
